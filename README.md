# 🚀 Manav Jain - Personal Portfolio

A modern, responsive, and interactive personal portfolio website designed to showcase my professional journey, technical skills, projects, achievements, and engineering interests.

The portfolio is built using HTML, CSS, and JavaScript with a unique PCB-inspired design theme that reflects my passion for electronics, technology, and engineering.

---

## 🌟 Project Overview

This portfolio serves as my professional digital identity and was developed during my internship to demonstrate front-end development skills and create a centralized platform for recruiters, professionals, and collaborators to explore my work.

The website is connected to my LinkedIn profile, allowing visitors to view my latest professional information and career progress.
